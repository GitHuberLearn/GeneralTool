## 项目开发

### 运行

```bash
- 1.  一般的用工具 Vscode 导入根目录建立项目使用 serve 启动
    Live Server › Settings: Multi Root Workspace Name (清空用了选择工作区)
- 2.安装sass:npm install -g sass
- 2.1  编译scss：sass xx.scss xx.css
- 2.2 监视编译：sass --watch xx.scss:xx.css
```
